# Security Policy

## Supported Versions

The following versions of the SwiftCart Food Delivery System are currently supported with security updates:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | ✅        |

## Reporting a Vulnerability

If you discover a security vulnerability, please follow the steps below to report it:

**Contact Us:** Report the vulnerability by emailing our security team at **francisoteng1.05@gmail.com**. Please provide as much information as possible, including:
* A detailed description of the vulnerability.
* Steps to reproduce the vulnerability.
* Any potential impact of the vulnerability.
* Suggested fixes, if any.

**Acknowledgment:** You will receive an acknowledgment of your report within 48 hours. Our security team will work to verify the vulnerability and determine its impact.

**Updates:** You will receive regular updates on the status of your report. We aim to provide updates at least once a week.

**Resolution:** If the vulnerability is accepted, we will work on a fix and release a security update. You will be credited for the discovery unless you wish to remain anonymous. If the vulnerability is declined, we will provide a detailed explanation.

**Disclosure:** To protect our users, we request that you do not disclose the vulnerability publicly until we have released a fix and informed our users.

Thank you for helping us keep the SwiftCart Food Delivery System secure.

For any other security-related questions or concerns, please contact us at **francisoteng1.05@gmail.com**.
